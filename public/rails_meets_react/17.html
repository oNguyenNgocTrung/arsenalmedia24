Responding to click events
To make the button at least slightly useful it would be good to do something when it is clicked.
React comes with a built-in system for detecting and handling events. It takes care of cross-browser
behavior and also makes testing easier.
To provide a component with a click handler function, you need to pass it as an argument to
onClick
property. A quick API reminder:
React.DOM.tagName(properties, children)
So far you have only used
properties
to provide HTML attributes such as
id
or
href
.
They are
also used for event handlers such as
onClick
and
onChange
.
