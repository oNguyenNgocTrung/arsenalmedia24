Serializing form data for AJAX forms
Since React is for the view purposes only, there is no built-in way to serialize form data for
making an AJAX call.
You can bind
onSubmit
handler in your form and get the real DOM form
element using
target
field inside the passed event. Then you can use one of the serialize methods
from jQuery -
serialize
or
serializeArray
or Prototype’s
serialize
. This is beyond the scope of the
