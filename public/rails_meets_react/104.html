Enabling touch events
React has built-in support for basic touch events (
onTouchCancel
,
onTouchEnd
,
onTouchMove
,
onTouchStart
). Since React 0.14 you don’t need to do anything to enable them.
If you want more support for touch events (like listening to touch gestures), you can integrate third-
party library like
Hammer.js
using lifecycle methods. Then you can bind events outside React using
ReactDOM.findDOMNode
. I’ll talk about it shortly.
Tip: Pass your handlers as properties to your component
It is often advisable to pass event handlers as a property to your component if you need to perform
a complex action which is beyond React control. It simplifies further change of such event handler
since you can change it in an initialization of the component and not within it directly.
