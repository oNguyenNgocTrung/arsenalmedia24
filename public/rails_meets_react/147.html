Testing through
React.findDOMNode
The most straightforward and ‘classical’ approach to test JavaScript is test against HTML that is
produced. Since it is the output a user is going to see, when you test it is correct you should have
confidence that everything works as expected.
We can follow this approach while testing React components. We have an access to DOM with
getDOMNode
function. All we need is to test it using any DOM querying technique you like -
I’m using
jQuery
because it is by default included in Rails and it’s really good in querying DOM
elements.
