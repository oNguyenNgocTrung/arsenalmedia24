From Rails partial to React
components
React is a technology which does have really smooth learning curve. You can easily transform your
simpler views to React components even without knowing what’s going on inside the components.
I want to show you an example.
React components are exactly like your view partials — but dynamic-ready.
You can easily
create a component to generate the same static HTML as your Rails partial, without even knowing
what’s going on inside the component. The syntax is really similar to what you can see in, for
example, Haml.
