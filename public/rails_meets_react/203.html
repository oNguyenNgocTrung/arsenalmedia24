Gulpfile.js
There is an equivalent of

for Gulp, called

. You need to create it manually:

Gulp operates on tasks. Task can be a function or an array of tasks to be performed sequentially.
You can run
gulp
without arguments and it will perform a
default
task, and
gulp <taskName>
to run a particular task.
