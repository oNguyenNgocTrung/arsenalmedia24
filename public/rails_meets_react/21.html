As you have probably already guessed, React will know how to render your element because you
defined that in the
render
method of the
OneTimeClickLink
component class.
But
React.createElement
is quite a verbose way to express your intention. Especially if you were
to use
OneTimeClickLink
elements multiple times. We are going to use
React.createFactory
instead.
oneTimeClickLink
=
React.createFactory(OneTimeClickLink)
$
->
ReactDOM.render(
oneTimeClickLink(),
document.getElementById(
"start"
)
)
React.createFactory
- returns a function that produces React Elements of a given type. In other
words:
oneTimeClickLink = (props, children) ->
React.createElement(OneTimeClickLink, props, children)
After this step the component can be used the same way as you used the basic
React.DOM.*
components so far. You could use it alongside them. Example:
React.DOM.div
className: "klass"
oneTimeClickLink()
React.DOM.span null, "Text inside"
oneTimeClickLink()
