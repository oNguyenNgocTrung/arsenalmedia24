Selecting meetup technology
We can create our programming meetups and invite people but we don’t know what technology we
will be discussing during the meetup. We already know how to use select boxes so implementing it
should not be hard. But there is one caveat. New technologies emerge quickly and we don’t want to
hardcode them. How do we pass data to our top-level component, especially from Rails? Usually we
do it with
props
. And here it is the same, but
react_ujs
will take care of starting the component
with the properties for us.
Let’s add technologies scaffold to our project.
bundle exec rails g scaffold technologies name:string --no-helper --no-stylesheets --no-te\
st-framework --no-javascripts --no-jbuilder
rm app/assets/stylesheets/scaffolds.scss
bundle exec rake db:migrate
Go to
http://localhost:3000/technologies
and add some technologies that you like ex.
Rails
and
React
.
