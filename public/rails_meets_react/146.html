jasmine-rails
:
Rails doesn’t come with any JavaScript testing environment by default. I recommend integrating
JavaScript development with
jasmine-rails
. Installation is pretty straightforward and consists of
following steps:
1. Add
jasmine-rails
gem to your Gemfile:
gem
'jasmine-rails'
,
group
:
:test
1. Run
bundle install
:
$
bundle install
1. Run Rails generator which creates environment (with directories) for you:
$
rails generate jasmine_rails:install
You put your JavaScript specs inside
spec/javascripts
with name
something_spec.js.coffee
.
jasmine-rails
leverages all features of sprockets - that means you can use whatever preprocessor
available, including
jsx
one. For more information about setting advanced options, visit
gem
repository at GitHub
. You’ll find fresh install instructions there too.
