The lifecycle methods are all about
when
you want to plug in your non-React integrations. Still, a
majority of the commonly used frontend libraries need an access to the real DOM.
React allows you to get an access to the generated DOM. If only your component is mounted you
can access its DOM representation using the
React.findDOMNode
function.
