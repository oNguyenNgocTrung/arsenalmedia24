The first argument of this method tells React what you want to render. The second argument

That’s your

document.getElementById
returns the
<div id="start">

React will render your
<div>
component
inside
the
<div id="start">
.
document.getElementById("start")
returns the
actual
HTML tag. That’s your
where

What you already learnt
You’ve just learned how to render a single HTML tag on your page using React. That’s an important
milestone. You know that when working with React you work with a representation of your HTML,
not with real HTML DOM nodes. The representation is called the
virtual DOM
.
React.DOM.div()
gives you one virtual tag. But at least once you need a real DOM node. It’s when you use
ReactDOM.render
to transform your virtual DOM into the real DOM in a browser. The place where
a real DOM tree (rendered by React) should be put must be a real DOM tag.
React transforms a virtual DOM tree into a real DOM tree
