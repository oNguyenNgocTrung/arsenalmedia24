SSE in Browser
So we’ve got the first render covered. But we need to make this component auto updates when the
data changes. We are going to use
Server Sent Events
for that. It’s a browser API for one way, server
to browser communication over HTTP connection. It even has automatic re-connections built-in.
ShowArticleFactory = React.createFactory(ShowArticle)
$ ->
$('[data-reactive]').each (_nop, element) ->
reactivePath = $(element).attr('data-reactive')
source = new EventSource(reactivePath);
source.addEventListener 'message', (e) ->
React.render(
ShowArticleFactory( JSON.parse(e.data) )
element
)
We look for
data-reactive
elements and make a connection to the URL. On event we ask react to
re-render a new version of the component in the same place.
