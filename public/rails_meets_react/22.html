To render a different content depending on whether your link was already clicked or not, a new
concept called
state
must be introduced. State is a piece of data that your React components store.
You can set it using
@setState(key:
value)
method. Then you can read the state of your
component using the
@state
field. React watches the component’s state and re-renders a virtual
DOM tree of the component whenever the state changes.
