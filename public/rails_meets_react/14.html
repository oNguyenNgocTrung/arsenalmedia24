React.DOM.tagName(properties, children)
Starting with
React.DOM.div({}, "Hello world!")
.
React.DOM
is a namespace for components
representing HTML5 tags. The most obvious one is
React.DOM.div
.
They are not real HTML5
tags. They are just their representation.
We can use them to describe what actual tags should be
rendered by React in the browser.
We use
React.DOM.div({},
"Hello
world!")
to create a virtual
<div>
tag without any
attributes and with
"Hello
world!"
as its inner HTML. When you use
React.DOM.tagName()
the first argument is for the component properties such as HTML attributes and the second one is
for its children. Remember:
React.DOM.tagName(properties, children)
You could render an empty
<div>
tag by using:

->

React.DOM.div
({})
,
document.getElementById
(
"start"
)
)
