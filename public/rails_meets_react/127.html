Flash notifications example
The flash notifications component is responsible for displaying notifications at the top of the
page–ones that you would normally set with
flash[:notice]
for example in your classic Rails
application. We needed such a component for a page on our website that was written in Single Page
Application style, to provide best speed for a ticketing company during crowded events for scanning
and selling tickets on the ground during an event. But we are only interested in the animated
notifications component that was used to display results of the operator actions.
.
Grab the example
cd reactjs-example
git checkout animations
bundle install
rails s
# visit http://localhost:3000/
