Summary
React is a great tool to have when it comes to building dynamic user interfaces. It is relatively easy
to learn and fast to start working with it. I hope this book gave you the knowledge you need to create
dynamic views of your applications - whether they are Rails applications or not. In Arkency working
with React gives us modularity and flexibility we desire. It was a fix to a problem with creating user
interfaces by hand, using DOM transformations (jQuery). It was an error-prone task that nobody
liked before. With React we are able to deliver our front-ends faster and with less bugs, without
tying with frameworks. We use
Hexagonal.js
approach to architecture our front-end applications,
and React fits well as an
adapter
to create the UI. We don’t need to change anything which is not
related to the UI - like external services (Facebook, Twitter, our Rails backend), integration and
business logic (use case and domain objects).
I hope this book convinced you that React is an easy choice if you need to add dynamism to your
apps. It’s easy to fall in love with React!
