Making it work
We have created modal and confirm dialog components. Now it’s time to make it work. We will
create a method that will render our confirm dialog and return a promise. Once the promise is
resolved or rejected, the dialog will be unmounted from DOM.
confirm = (message, options = {}) ->
props = $.extend({message: message}, options)
wrapper = document.body.appendChild(document.createElement('div'))
component = React.render(React.createElement(Confirm, props), wrapper)
cleanup = ->
React.unmountComponentAtNode(wrapper)
setTimeout -> wrapper.remove()
component.promise.always(cleanup).promise()
When resolving or rejecting a promise, we are unmounting the whole
Confirm
component to
cleanup the DOM (I prefer removing nodes from DOM than just hiding them via CSS). We are
also removing the wrapper node since it’s not needed anymore after the dialog is closed - each time
we call the
confirm
method, new wrapper node would be created and added to DOM (you may
also create a confirm target node upfront - then you would only need to mount and unmount the
component in
confirm
).
