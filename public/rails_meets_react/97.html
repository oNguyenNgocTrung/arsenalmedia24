Integration with the external world
State changes come to React components from many possible sources. You may be interested in
how you can connect those sources with your components. React provides quite a big variety of
techniques to interact with the external world. For user-generated events (like clicks, input changes
or hovering over elements, not to mention drag-and-drop and touch events) React comes with the
special properties to handle such events. When you output data from your component you can use
managed input components
to do so in an easy way. If you want to modify behaviour or look of
the component via an external library or perform actions which do not fit to
render
well, you can
use lifecycle methods. They act as “hooks” to certain events during React component’s life.
