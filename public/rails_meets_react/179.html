What we learned in this small comparison example?
• You create new react component with

• The most important method that your component must implement is

. This method is
responsible for returning the structure of HTML that we want to render. The returned data is
called
virtual
DOM
because react keeps its own version of DOM tree and uses it to render
actual DOM elements in the browser.
•

method can use existing react components available under
React.DOM
scope. These
are elements such as
React.DOM.div, React.DOM.span
, etc. You can use them as building
blocks to create your own components.
• You can pass html properties as first argument when using react building blocks

• You can use second argument to provide text of the element
